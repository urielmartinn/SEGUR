Aurrebaldintzak
- Docker
- Docker Compose


1. Repositorioa klonatu
   git clone https://github.com/urielmartinn/SEGUR.git
   cd REPO_IZENA

3. Entregaren adarrera aldatzeko:
   git checkout zeregina_1

4. Sortu eta abiarazi
   docker-compose up --build -d

5. Nabigatzailetik sartzeko:
   http://localhost:81/
